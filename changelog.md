# ASP34 Report System — История версий

---

## v3.2 (05.06.2026) — Текущая версия

### 🆕 Новые возможности
- **Уведомление об обновлении на почту** — каждый робот присылает письмо после автообновления
- **Backup с версией в имени файла** — старое: `script.backup`, новое: `script.v3.1.backup`
- **Улучшенная функция автообновления** — скачивание, проверка, установка, уведомление

### 🔧 Улучшения
- Email-уведомления теперь на двух языках (RU/EN)
- Вся документация переведена на английский
- Добавлен установщик задач `install_tasks.ps1`
- Все компоненты подписаны `ASP34 SOFT 2026`

## v3.1 (04.06.2026)

### 🆕 Новые возможности
- **Автообновление с GitHub Pages** — роботы проверяют обновления каждую ночь в 03:00
- **Backup перед обновлением** — старая версия сохраняется автоматически
- **Сохранение оригинальной даты файла** — формат: `AZS_Station_26.06.03_08-15-30.xml`

### 🔧 Исправления
- Исправлена кодировка логов (UTF-8)
- Исправлена ошибка отправки почты (неправильный sender_email)
- Исправлены кракозябры в именах АЗС
- Исправлен двойной слеш в пути Export

## v3.0 (03.06.2026)

### 🆕 Новые возможности
- **Настройки в отдельном файле** `config_azs.json` — при обновлении настройки не теряются
- **Сервер обновлений на GitHub Pages** — https://adminvtk34.github.io/azs-updates/
- **Веб-панель** с информацией о версиях и ссылками для скачивания

### 🔧 Улучшения
- Код и настройки разделены (безопасность, удобство)
- Поддержка любого количества АЗС
- Установщик `install.bat` — установка за 5 минут
- Тестовый скрипт `test_mail.ps1` для проверки почты

## v2.0 (15.05.2026)

### 🆕 Новые возможности
- **Три режима работы** — активный (07:30-09:30), пассивный, экстренный
- **Защита после перезагрузки** — задержка перед началом работы
- **Обработка нескольких файлов** одновременно
- **Ожидание разблокировки файла** — если Topaz ещё пишет

### 🔧 Исправления
- Устранена утечка памяти при ошибках SMTP
- Исправлена ошибка с сохранением pending файлов
- Автоочистка логов и архива старше 30 дней
